* Python static-site build pipeline (`build.py`)
* Jinja2 templates in `templates/`
* Markdown content with YAML frontmatter for blog, wiki, and paper pages
* Python-Markdown extensions, including fenced code, tables, `codehilite`, and `pymdownx.arithmatex`
* MathJax loaded on wiki and paper pages for TeX rendering
* Makefile tasks for building the site and scaffolding new content
